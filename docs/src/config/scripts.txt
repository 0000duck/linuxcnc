= Scripts

== Info

Several scripts are included with LinuxCNC to do various functions not normally
done with LinuxCNC GUI's.

== Axis Remote

The axis-remote script triggers commands in a running Axis GUI.

----
axis-remote --clear|--ping|--reload|--quit|--mdi command|filename
axis-remote -c|-p|-r|-q|-m command|filename
----

* --clear clears the backplot
* --ping checks to see if Axis is running. Returns 0 if true and 1 if false
* --reload reload the current G code file
* --quit terminates Axis
* --mdi command executes a mdi command
* file name loads a file into Axis

.Example Bash Script to Load a File
----
if axis-remote -p;
then axis-remote the-file-i-want;
fi
----

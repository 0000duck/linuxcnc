= Configuration rapide pour moteurs pas à pas

[[cha:configuration_rapide_steppers]] (((Stepper Quickstart)))

Cette section suppose qu'une installation du logiciel à partir du CD Live a
été faite. Après cette installation et avant de continuer, il est recommandé
de connecter le PC sur Internet pour y faire les dernières mises à
jour. Pour les installations plus complexes se référer au Manuel de
l'intégrateur.

== Test de latence (Latency Test)

Le test de latence détermine la capacité du processeur à répondre aux
requêtes qui lui sont faites. Certains matériels peuvent interrompre ce
processus, causant des pertes de pas lorsque le PC pilote une machine
CNC. Ce test est la toute première chose à faire pour valider un PC.  Pour
le lancer, suivre les instructions de la section <<cha:test-de-latence,sur
le test de latence>>.

[[sec:Sherline]]

== Sherline (((Sherline)))

Si la machine est une Sherline, plusieurs configurations prédéfinies sont
fournies.  Au premier démarrage de LinuxCNC, le sélecteur de configuration
s'ouvre, sélectionner alors le modèle correspondant à la machine _Sherline_,
puis accepter d'enregistrer une copie.

[[sec:Xylotex]]

== Xylotex (((Xylotex)))

Si la machine est une Xylotex, utiliser l'assistant graphique de
configuration fourni avec LinuxCNC pour créer rapidement une configuration
personnalisée <<cha:Assistant-graphique-StepConf,avec l'assistant
Stepconf>>.

== Informations relatives à la mécanique

But, regrouper les informations à propos des réducteurs.  Utilisées pour
définir la taille d'un pas dans l'unité utilisateur. La taille du pas est
utilisée par SCALE dans le fichier .ini.

[width="90%", options="header"]
|====================================================================
|Axes | Pas par tour | Micropas | Dents moteur | Dents vis | Pas de la vis
|X    |              |          |              |           |
|Y    |              |          |              |           |
|Z    |              |          |              |           |
|     |              |          |              |           |
|====================================================================

* 'Pas par tour' - indique combien de pas moteur sont nécessaires pour que
celui-ci fasse un tour. Valeur typique: 200.

* 'Micro-pas' - indique combien d'impulsions le pilote doit recevoir pour que le
moteur tourne d'un angle équivalent à un pas.
Si les micro-pas ne sont pas utilisés, cette valeur devra être mise à 1.
Si les micro-pas sont utilisés, les valeurs dépendent du pilote matériel du moteur.

* 'Dents moteur et Dents vis' - Valeur de réduction si une réduction
(poulies/courroie, chaine, pignons etc) entre le moteur et la vis est utilisée.
Sinon mettre 1 pour les deux paramètres.

* 'Pas de la vis' - indique la longueur de déplacement du mobile pour un tour
de la vis d’entraînement de l'axe.

Le résultat qui est recherché, c'est combien de pas le pilote devra fournir
pour que le mobile avance d'une unité utilisateur (pouces ou mm).


.Unités en pouces
====
....
Moteur            = 200 pas par tour
Pilote            =  10 micropas par pas
Dents côté moteur =  20
Dents côté vis    =  40
Pas de vis        =   0.2000 pouces par tour
....
====


D'après l'exemple ci-dessus  (pouces):

 - le mobile se déplacera de 0.200 pouces pour 1 tour de vis.
 - Le pilote demandera 10 impulsions en entrée pour faire un pas entier.
 - Le moteur fera 2000 micro-pas par tour de vis.
 - Le pilote recevra 2000 impulsions de pas pour faire tourner le moteur d'un tour.

Ainsi, l'échelle nécessaire sera (pouces):

image::images/step-calc-inch-math.png[align="left"]


////
latexmath:[ \frac{200 motor steps}{1 motor rev} \times
\frac{10 microsteps}{1 motor step} \times
\frac{2 motor revs}{1 leadscrew rev} \times
\frac{1 leadscrew revs}{0.2000 inch}
= \frac{20,000 microsteps}{inch} ]
////


.Unités en millimètres
====
....
Pas par tour      = 200 pas par tour
Micropas          =   8 micropas
Dents côté moteur =  30
Dents côté vis    =  90
Pas de la vis     =   5.00 mm par tour
....
====


D'après l'exemple ci-dessus (mm):

 - la vis déplacera le mobile de 5.00 mm par tour.
 - Le moteur fera 3 tours pour 1 tour de vis. (90/30)
 - Le pilote utilisera 8 micro-pas pour faire un pas.
 - Le pilote aura besoin de 1600 impulsions pour un tour moteur.

Ainsi, l'échelle nécessaire sera (mm):

image::images/step-calc-mm-math.png[align="left"]


////
latexmath:[ \frac{200 motor steps}{1 motor rev} \times
\frac{8 microsteps}{1 motor step} \times
\frac{3 motor revs}{1 leadscrew rev} \times
\frac{1 leadscrew revs}{5.000 mm}
= \frac{960 microsteps}{mm} ]
////
